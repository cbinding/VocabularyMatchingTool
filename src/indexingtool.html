AAT Indexing Tool

The AAT Indexing Tool utilizes the Getty Vocabulary Program SPARQL endpoint to provide a search and browsing facility on the Getty Art & Architecture Thesaurus (AAT) for specific form fields. The tool can be integrated into existing data entry forms to allow data records to be indexed using Getty AAT terms.

In the (live) example below the first two form fields are INPUT elements, the second two are DIV elements. Clicking on a field brings up the AAT Indexing tool at that location allowing AAT search (e.g. try searching for 'axe' or 'brooch'). Selecting any of the result concepts allows browsing of the AAT concept structure. The currently selected concept is shown near the bottom of the control. Once a suitable concept has been selected click 'save' to populate the original field with the selection. In the case of simple text input fields the currently selected term is used to populate the field as text. In the case of the DIV fields, the term plus a URI are used to populate the field as a link to the selected AAT concept.

Example


[subject]
[subject]

Form fields can then be populated with AAT terms. To deploy this on your own web page download the indexing tool stylesheet (usw.aatindexingtool.css) and latest minified script (usw.aatindexingtool.yyyymmdd-min.js) from https://github.com/cbinding/VocabularyMatchingTool. Include links to jquery and jquery-ui scripts (these are dependencies), and also links to the downloaded files within the head of your page: 

<html>
    <head>
        <script src="//code.jquery.com/jquery-1.9.1.min.js"></script>
        <script src="//code.jquery.com/ui/1.10.3/jquery-ui.js"></script>
        
        <link rel="stylesheet" href="css/usw.aatindexingtool.css">
        <script src="lib/usw.aatindexingtool.20150428-min.js"></script>
    </head>
</html>

Next add the class name 'usw-aatindexingtool' to any fields requiring AAT term/concept lookup. If in doubt just right click and 'view source' on this page to see how it has been done:

<div>
    <label for="subject">subject:</label>
    <input type="text" id="subject" size=25 class="usw-aatindexingtool" value=""/>
</div>

That's it. Clicking on that field will now pop up the AAT Indexing Tool at that location, allowing the field to be populated with the chosen AAT term.
